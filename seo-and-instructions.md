## GoHighLevel Paste Instructions

Follow these steps to deploy your new landing page in GoHighLevel:

1. **Create a new Funnel or Website Page** in GHL.
2. Add a **Full Width Section** (ensure "Allow Rows to take entire width" is toggled ON).
3. **Remove ALL padding** on the Section, Row, and Column (set top, bottom, left, right padding to 0).
4. Add a single Row with one Column.
5. Drop in a **Custom Code / HTML** element.
6. Open the code editor and **paste the entire contents** of the provided `landing-page.html` file.
7. *Important:* Search the code for `FORM_ID_HERE` and replace it with your actual GHL Form ID so the popup works correctly.
8. Save and Preview!
